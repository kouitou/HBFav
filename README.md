はてなブックマークお気に入りリーダー
=====

はてなブックマークのお気に入りをTwitterのタイムラインみたいに読みたい。
お気に入りは、follow しているユーザーにフォーカスがあたっている方が個人的に好み

* ./ : Titanium による iOS クライアント
* server/ : RSS を JSON に変換するサーバー (node)
